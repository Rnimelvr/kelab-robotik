# Kelab Robotik KV Sepang (HTML + CSS)

Portal maklumat rasmi dan borang pendaftaran ahli bagi Kelab Robotik KV Sepang. Menyokong:
- Halaman Utama (Maklumat Kelab)
- Borang Pendaftaran (Nama, Email, Kelas)
- Paparan Senarai Ahli (Jadual Keahlian)

## Cara guna

1. Buka fail `index.html` dalam pelayar web untuk melihat halaman utama.
2. Pergi ke halaman `daftar.html` untuk mengisi maklumat ahli baharu.
3. Buka fail `senarai.html` untuk melihat senarai ahli yang telah berdaftar.
